# 🚀 Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/vital-stats.git
cd vital-stats
```

---

## 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 3️⃣ Run Application

```bash
python monitor.py
```

---

# 🔨 Build EXE File

## Install PyInstaller

```bash
pip install pyinstaller
```

---

## Build EXE

```bash
.\build_exe.bat
```

---

## EXE Output

```text
dist/monitor.exe
```
